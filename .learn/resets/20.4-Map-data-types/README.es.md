# `20.4` Map data types

## 📝 Instrucciones:

1. Modifica la función de `array.map()` para que cree un nuevo arreglo que contenga los tipos de datos de cada elemento del arreglo dado.

## 💡 Pista:

+ Usa la función `typeof` para obtener el tipo de dato.

## 💻 Resultado esperado:

```js
[ 
  'string', 'string', 
  'string', 'string', 
  'string', 'string', 
  'number', 'number' 
]
```
